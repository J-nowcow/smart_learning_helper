# 📚 스마트 학습 도우미

온라인 강의 사이트에서 강의를 자동으로 연속 학습하고, **STT(Speech-to-Text) 기능**으로 강의 내용을 텍스트로 추출하는 프로그램입니다.

## 🚀 사용 방법

### 실행파일 사용 (추천)
1. **`python build_executable.py`** 실행
2. 생성된 **`스마트_학습_도우미_v1.0`** 폴더를 압축해서 배포
3. 사용자는 **`스마트_학습_도우미`** 더블클릭으로 실행

## 📋 필요 사항
- **Chrome 브라우저** 설치 필요
- **인터넷 연결** 필요
- **STT 기능 사용 시**: 추가 라이브러리 설치 필요

## 🎬 프로그램 사용법
1. 프로그램 실행
2. **STT 텍스트 추출** 체크박스 선택 (선택사항)
3. STT 모델 선택 (tiny, base, small, medium, large)
4. 브라우저에서 온라인 강의 로그인
5. GUI에서 "로그인 완료" 버튼 클릭
6. 학습 시작!

## 🎤 STT 기능
- **Whisper AI** 기반 음성 인식
- **한국어 지원** 우수
- **오프라인 처리** 가능
- **자동 텍스트 추출**: 강의 완료 후 자동으로 텍스트 파일 생성
- **다양한 출력 형식**: TXT, SRT 자막 파일

## ⚠️ 주의사항
- 강의가 100% 재생된 후 10초 버퍼를 두고 다음 강의로 이동
- 네트워크 연결이 불안정하면 일시정지될 수 있음
- 브라우저를 닫으면 학습이 중단됨
- **STT 기능**: 첫 실행 시 모델 다운로드로 시간이 걸릴 수 있음

## 📁 파일 구성
- `ktedu_gui.py` - 메인 GUI 프로그램
- `ktedu_auto_player.py` - 학습 엔진 (리팩토링됨)
- `browser_manager.py` - 브라우저 관리 모듈
- `video_player.py` - 동영상 플레이어 모듈
- `video_stt.py` - STT 처리 모듈
- `chromedriver_140` - Chrome 드라이버
- `requirements.txt` - 필요한 패키지 목록
- `build_executable.py` - 실행파일 빌드 스크립트

## 🔧 STT 모델 크기별 특징
- **tiny**: 39MB, 빠른 처리, 낮은 정확도
- **base**: 74MB, 균형잡힌 성능 (추천)
- **small**: 244MB, 높은 정확도
- **medium**: 769MB, 매우 높은 정확도
- **large**: 3GB, 최고 정확도 (10GB+ RAM 필요)
